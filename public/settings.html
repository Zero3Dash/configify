🗓️ Compliance Schedules

Automatically run compliance checks on a recurring interval. Schedules run server-side; the server checks for due jobs every 60 seconds.

⚠️ You need admin role to create or modify schedules.
Each schedule checks either a specific golden config against all its assigned devices, or runs a full audit across every assignment. Results appear automatically in the Compliance dashboard.
